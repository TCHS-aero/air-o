## Getting Started

### Prerequisites
1. Python 3.x installed on your system.
2. A Discord account and bot token.
3. Basic familiarity with Python and Discord API.

<a name="installation">

---

## Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/TCHS-aero/air-o.git && cd air-o
    ```

2. Install the required dependencies:
    ```bash
    pip install -r requirements.txt
    ```

3. Set your bot token:
    Create a new file in the home directory of this repository called `token.env`. Replace YOUR_BOT_TOKEN with your actual application token.
   ```env
   TOKEN=YOUR_BOT_TOKEN
   ```

<a name="usage">

---

## Usage

Run the bot:
```bash
python air-o.py
```
Make sure your bot token is correctly configured to connect to the Discord API.
